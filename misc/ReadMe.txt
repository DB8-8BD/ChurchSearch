ShowMe sample web application
Description:
ShowMe is a simple app that shows you how to process form fields submitted from a webpage.

Requirements for this wizard:
-Must be running Windows 10
-Must have Windows 10 SDK 14393 installed
-Must have Visual Studio 2015 w/ update 3 or Visual Studio 2017

Debugging:
You may initiate remote debugging by pressing F5 if:
-The raspberry pi should be running the latest version of Windows Iot Core
-Must have w3pi copied to c:\test directory on device
-MSVSMON is running on your raspberry pi
-if you have named your raspberry pi minwinpc. 
After the debugger has attached to the remote process, open up your browser, and browse to http://minwinpc/atlserver/showme.srf
Otherwise, refer to the w3pi's on-line help manual for assistance with remote debugging. You can also view the getting-started guide included with the extension.
